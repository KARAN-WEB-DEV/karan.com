CSS Introduction

Cascading Style Sheets (CSS) is a styling language used for styling a web page. It is a language that specifies how web documents (HTML, XML) are presented to users.

CSS can change the color, size, and style of text, backgrounds, and links. It is indispensable for creating interesting layouts and enhancing user experience.

For example, it can be used to create sidebars and ease navigation, among many other things. Hence, CSS is an essential component of web development, alongside HTML and JavaScript

CSS Syntax

The basic syntax of CSS includes 3 main parts:

  • Selectors- specifies the HTML element that we want to apply the styles
  • Property-specifies the attribute of HTML elements that we want to change (color, background, and so on)
  • value-specifies the new value you want to assign to the property (color of the text to red, background to gray, and so on)

Let's take a example of css

                            <!DOCTYPE html>
                            <html lang="en">
                                    <head>
                                            <title>css example</title>
                                            <style>
                                                 p{
                                                    color:blue;
                                                 }
                                             </style>
                                    </head>
                                    <body>
                                            <p>This is a simple text.</p>
                                    </body>
                            </html>

comments in css

" /* this is comment in css */"

is a single line comment. It can be seen while reading the code but gets ignored by the browser during the rendering of the page.


We can also add a multiline comment on CSS. Simply by ending the comment in a different line.



Why we use comments in css?

  • Comments in code help the person reading the code understand what you were trying to do when you wrote it.

  • This makes it easier for other developers to understand the code and make changes if necessary.

  • Comments are also useful for anyone who needs to maintain the code in the future.

Include CSS in a webpage

CSS is used for styling the look and formatting of a document written in HTML. There are three ways to add CSS in HTML

  • Inline CSS:Styles added directly to the HTML element.
  • Internal CSS:Styles defined at the head section of the document.
  • External CSS:tyles defined in a separate file.

Inline Style

Inline style is the approach of adding CSS rules directly to the HTML element using the style attribute


                            <!DOCTYPE html>
                            <html lang="en">
                                    <head>
                                            <title>Browser</title>
                                            
                                    </head>
                                    <body>
                                            <h1>This is h1.</h1>
                                            <p>This paragraph doesn't have CSS..</p>
                                            <p style="color:red">This paragraph is styled with inline CSS.</p>
                                    </body>
                            </html>

Internal CSS

Internal CSS applies CSS styles to a specific HTML document. Internal CSS is defined inside an HTML document


                                <!DOCTYPE html>
                                <html lang="en">
                                        <head>
                                        <style>
                                        p{
                                           color:blue;
                                        }
                                    </style>
                                                <title>Browser</title>
                                                
                                        </head>
                                        <body>
                                                <h1>internal css example</h1>
                                                <p>This is paragraph1.</p>
                                                <p>This is paragraph2.</p>
                                                <div>This is content inside a div.</div>
                                        </body>
                                </html>
